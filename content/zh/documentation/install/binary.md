预编译的二进制文件 [`pyALPS`](https://pypi.org/project/pyalps/) 支持大多数 Linux 和 macOS 系统。可通过 pip 包管理器安装：

    pip install pyalps

请确保您的 Python 版本 ≥ 3.9。

## 安装视频指南

Windows 用户可选择以下任一方式：
1. 先安装 Linux 子系统再安装 ALPS
2. 在虚拟箱环境中安装 ALPS
