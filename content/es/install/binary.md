El binario precompilado [`pyALPS`](https://pypi.org/project/pyalps/) se puede instalar en la mayoría de las máquinas Linux y MacOS. `pyALPS` se puede instalar usando el gestor de paquetes de Python pip:

    pip install pyalps

Asegúrate de que tu versión de Python sea Python >= 3.9.
Ten en cuenta que la versión binaria de ALPS no admite la ejecución paralela de los códigos. Para una versión paralela de ALPS, por favor usa las instalaciones desde [fuente](../source) o con [spack](../spack).
