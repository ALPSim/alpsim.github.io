## Gaps de espín en sistemas de espín cuánticos 1D

En este tutorial usaremos el programa de diagonalización dispersa para calcular gaps de espín en sistemas de espín cuánticos 1D y estudiar su escalamiento de tamaño finito.
Una pregunta central para cualquier cadena de espín antiferromagnética es si tiene un gap de excitación finito por encima de su estado fundamental, o si carece de gap. Haldane conjeturó —y hoy en día está firmemente establecido, tanto numéricamente como mediante argumentos de teoría de campos— que las cadenas de Heisenberg isótropas se comportan de manera completamente distinta según el número cuántico de espín $S$ sea entero o semientero: las cadenas de espín entero tienen un estado fundamental único y con gap (el "gap de Haldane"), mientras que las cadenas de espín semientero no tienen gap y presentan correlaciones que decaen como ley de potencias ([F.D.M. Haldane, Phys. Rev. Lett. 50, 1153 (1983)](https://doi.org/10.1103/PhysRevLett.50.1153)). Dado que un sistema finito siempre tiene un espectro discreto, esta diferencia cualitativa solo se hace visible al extrapolar el gap de tamaño finito a $L\to\infty$ —que es exactamente lo que haremos a continuación para $S=1$ y $S=1/2$.

El hamiltoniano en ambos casos es la cadena de Heisenberg isótropa

$$
H = J \sum_{\langle i,j \rangle} \mathbf{S}_i \cdot \mathbf{S}_j , \qquad J=1 ,
$$

en una cadena periódica de $L$ sitios, que difieren solo en la longitud de espín local $S$.

### Parámetros

| Parámetro | Significado | Valor |
|---|---|---|
| `LATTICE` | cadena periódica | `chain lattice` |
| `MODEL` | modelo de espín cuántico | `spin` |
| `local_S` | número cuántico de espín por sitio | 1 (`parm2a`) o 1/2 (`parm2b`) |
| `J` | acoplamiento de Heisenberg a primeros vecinos | 1 |
| `L` | longitud de la cadena | 4, 6, 8, 10 |
| `CONSERVED_QUANTUMNUMBERS` | simetría usada para diagonalizar $H$ en bloques | `Sz` |
| `Sz_total` | restringe la diagonalización a un único sector $S_z$ | 0 (sector singlete) y 1 (sector triplete) |

### Red

Ambas cadenas usan la `chain lattice` periódica incorporada de la [biblioteca de redes de ALPS](../../../documentation/intro/latticehowtos), esbozada aquí para $L=6$:

```
    J     J     J     J     J
o-------o-------o-------o-------o-------o
0       1       2       3       4       5
|_______________________________________|
                   J   (bond 5-0, periodic)
```

### Método

Diagonalizar por separado en los sectores $S_z=0$ y $S_z=1$ y tomar el autovalor más bajo de cada uno aísla el estado fundamental singlete y la excitación triplete más baja sin tener que buscar en todo el espectro (mucho más grande). Incluso el sector más grande aquí ($S=1$, $L=10$, dimensión $8953$) es manejado fácilmente por la iteración de Lanczos que hay detrás de `sparsediag`; la diagonalización completa sería innecesaria y considerablemente más costosa, ya que solo necesitamos el estado más bajo en cada sector.

### Gap de espín de una cadena de espín 1

#### Usando la línea de comandos
El archivo de parámetros <a href="../codes/ed-02-gaps/parm2a" download>`parm2a`</a> configura la diagonalización exacta de la cadena cuántica S=1 con 4 a 10 sitios en los sectores singlete y triplete

```
MODEL="spin"
LATTICE="chain lattice"
CONSERVED_QUANTUMNUMBERS="Sz"
local_S=1
J=1
Sz_total=0
{L=4}
{L=6}
{L=8}
{L=10}
Sz_total=1
{L=4}
{L=6}
{L=8}
{L=10}
```
    
El parámetro Sz_total restringe el número cuántico conservado Sz al valor dado. Consulte la documentación de la biblioteca de modelos para más detalles sobre cómo configurar modelos y números cuánticos conservados.
Usando la secuencia estándar de comandos, primero puede convertir los parámetros de entrada a XML y luego ejecutar la aplicación `sparsediag`:

```
parameter2xml parm2a
sparsediag --write-xml parm2a.in.xml
```
    
La evaluación de los gaps puede hacerse ahora manualmente examinando los archivos de salida, o podemos usar Python para automatizar todo el flujo de trabajo.

#### Usando Python
Para configurar y ejecutar la simulación en Python usamos el script <a href="../codes/ed-02-gaps/tutorial2a.py" download>`tutorial2a.py`</a>. Las primeras partes de este script importan los módulos requeridos, preparan los archivos de entrada como una lista de diccionarios de Python, escriben los archivos de entrada y ejecutan la aplicación

```
import pyalps
import numpy as np
import matplotlib.pyplot as plt
import pyalps.pyplot
parms = []
for l in [4, 6, 8, 10]:
    for sz in [0, 1]:
        parms.append({ 
            'LATTICE'                   : "chain lattice", 
            'MODEL'                     : "spin",
            'local_S'                   : 1,
            'J'                         : 1,
            'L'                         : l,
            'CONSERVED_QUANTUMNUMBERS'  : 'Sz',
            'Sz_total'                  : sz
        })
     
input_file = pyalps.writeInputFiles('parm2a',parms)
res = pyalps.runApplication('sparsediag',input_file)
```
    
Para ejecutarlo, escriba `python tutorial2a.py` en su terminal. Ahora tenemos los mismos archivos de salida que en la versión de línea de comandos.
A continuación cargamos los espectros de cada uno de los tamaños de sistema y sectores de espín:

```
data = pyalps.loadSpectra(pyalps.getResultFiles(prefix='parm2a'))
```
    
Para extraer los gaps necesitamos escribir unas pocas líneas de Python, para establecer una lista de longitudes y un diccionario de Python con la energía mínima en cada sector (L,Sz):

```
lengths = []
min_energies = {}
for sim in data:
    l = int(sim[0].props['L'])
    if l not in lengths: lengths.append(l)
    sz = int(sim[0].props['Sz_total'])
    all_energies = []
    for sec in sim:
        all_energies += list(sec.y)
    min_energies[(l,sz)]= np.min(all_energies)
```

Y finalmente hacemos un gráfico del gap en función de 1/L y lo mostramos
    
```
gapplot = pyalps.DataSet()
gapplot.x = 1./np.sort(lengths)
gapplot.y = [min_energies[(l,1)] -min_energies[(l,0)] for l in np.sort(lengths)]  
gapplot.props['xlabel']='$1/L$'
gapplot.props['ylabel']='Triplet gap $\Delta/J$'
gapplot.props['label']='S=1'

plt.figure()
pyalps.plot.plot(gapplot)
plt.legend()
plt.xlim(0,0.25)
plt.ylim(0,1.0)
plt.show()
```

### Datos de salida

La siguiente tabla enumera el gap triplete exacto $\Delta(L)=E_0(S_z=1)-E_0(S_z=0)$ obtenido a partir de los dos archivos de parámetros, como referencia para comparar con su propia ejecución:

| $L$ | $1/L$ | Gap $\Delta/J$, S=1 | Gap $\Delta/J$, S=1/2 |
|---|---|---|---|
| 4  | 0.250 | 1.0000 | 1.0000 |

| 8  | 0.125 | 0.5936 | 0.5227 |
| 10 | 0.100 | 0.5248 | 0.4232 |

El gap de S=1 se está estabilizando y se extrapola (por ejemplo, mediante un ajuste cuadrático en $1/L$) a un valor finito cercano a la estimación de la literatura para el gap de Haldane, $\Delta_\infty \approx 0.41\,J$, mientras que el gap de S=1/2 sigue disminuyendo y es consistente con una extrapolación a cero —el sello distintivo de una cadena sin gap.

## Resumen

Para el mismo acoplamiento de Heisenberg antiferromagnético a primeros vecinos, la cadena S=1 se extrapola a un gap de excitación finito mientras que la cadena S=1/2 se extrapola a cero, confirmando directamente la conjetura de Haldane en cadenas lo bastante pequeñas como para diagonalizarlas exactamente.
