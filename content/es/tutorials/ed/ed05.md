## Punto crítico de la cadena de Heisenberg con interacción a segundos vecinos

En este tutorial, continuaremos con la última parte del tutorial ED-04, donde se consideró la cadena de Heisenberg. Añadiremos un término de acoplamiento a segundos vecinos $J_2 \sum_{\langle \langle i,j \rangle \rangle} S_i \cdot S_j$ al hamiltoniano.

En el límite $J_2 = 0$, este modelo se reduce a la cadena de Heisenberg crítica, que es soluble mediante el ansatz de Bethe. En $J_2/J_1=0.5$, el modelo también es soluble. Resulta que el estado fundamental es
$$
|\Psi\rangle = \left(|\uparrow\rangle_1 |\downarrow\rangle_2 - |\downarrow\rangle_1 |\uparrow\rangle_2\right) (|\uparrow\rangle_3 |\downarrow\rangle_4 - |\downarrow\rangle_3 |\uparrow\rangle_4) (|\uparrow\rangle_5 |\downarrow\rangle_6 - |\downarrow\rangle_5 |\uparrow\rangle_6)
$$

Esto es, por supuesto, [indicio de una transición de fase](https://doi.org/10.1103/PhysRevE.76.061108) en algún $J_1/J_2$ intermedio \in $(0,1/2)$.

En la primera parte de este tutorial, localizaremos la posición del punto crítico observando cómo cambia el espectro, en particular el gap en los distintos sectores de simetría, a medida que ajustamos los acoplamientos. En la segunda parte, revisaremos el contenido de TCC de la cadena crítica. Analíticamente, se puede demostrar que el modelo en el punto crítico está descrito por la misma TCC que la cadena de Heisenberg, pero el peso del operador marginal que provoca las correcciones de tamaño finito que se anulan logarítmicamente es cero, por lo que las dimensiones de escala pueden encontrarse con mucha mayor precisión.

Entonces, primero, grafiquemos la energía del estado fundamental y del primer estado excitado, así como el gap en el sector singlete ($S_z = 0$) y triplete ($S_z=1$).

### Python

Comenzamos con las importaciones habituales:

```
import pyalps
import pyalps.plot
from pyalps.dict_intersect import dict_intersect
import numpy as np
import matplotlib.pyplot as plt
import copy
import math
```
    
De nuevo, usamos el número cuántico $S_z$, pero ahora ejecutaremos simulaciones en distintos sectores $(S_z=0,1)$. Ejecutamos para los tamaños de sistema $L=6,8$, ya que el efecto que buscamos ocurre ya en tamaños de sistema muy pequeños.

```
prefix = 'alps-nnn-heisenberg'
parms = []
for L in [6,8]:
    for Szt in [0,1]:
        for J1 in np.linspace(0,0.5,6):
            parms.append({
                'LATTICE'              : "nnn chain lattice",
                'MODEL'                : "spin",
                'local_S'              : 0.5,
                'J'                    : 1,
                'NUMBER_EIGENVALUES'   : 2,
                'CONSERVED_QUANTUMNUMBERS' : 'Sz',
                'Sz_total'             : Szt,
                'J1'                   : J1,
                'L'                    : L
            })

input_file = pyalps.writeInputFiles(prefix,parms)
res = pyalps.runApplication('sparsediag', input_file)
# res = pyalps.runApplication('sparsediag', input_file, MPI=4)
data = pyalps.loadEigenstateMeasurements(pyalps.getResultFiles(prefix=prefix))
```

El análisis de datos es en este caso un poco más laborioso que en los anteriores. En particular, dependeremos en gran medida de la funcionalidad de conjuntos de datos jerárquicos. Para entender la física, en realidad basta con observar solo el estado fundamental y el primer excitado, así que si le parece que el cálculo de los gaps es demasiado confuso, no se preocupe demasiado por ello.

En un primer paso, unimos todas las energías para un conjunto dado de J1, L, Sz_total y las ordenamos. Primero, agrupamos por los parámetros J1, L, Sz_total. Cada elemento en el bucle sobre los grupos contendrá entonces una lista de conjuntos de datos para distintos momentos. Los uniremos y luego usaremos la función `dict_intersect` para encontrar las propiedades del conjunto de datos resultante; esta función simplemente toma una lista de diccionarios y devuelve la parte que es igual para todos ellos. Usamos la función `argsort` de numpy para obtener la lista de índices que ordenará y; esto nos permite ordenar x en consecuencia, aunque probablemente no sea necesario.

```
grouped = pyalps.groupSets(pyalps.flatten(data), ['J1', 'L', 'Sz_total'])
nd = []
for group in grouped:
    ally = []
    allx = []
    for q in group:
        ally += list(q.y)
        allx += list(q.x)
    r = pyalps.DataSet()
    sel = np.argsort(ally)
    r.y = np.array(ally)[sel]
    r.x = np.array(allx)[sel]
    r.props = dict_intersect([q.props for q in group])
    nd.append( r )
data = nd
```

A continuación, tenemos que eliminar los estados que aparecen en el sector $S_z=1$ del sector $S_z=0$. Los agrupamos por J1, L, de modo que cada grupo contenga los espectros de los dos sectores distintos de Sz_total. Luego usamos la función `subtract_spectrum`, que elimina del conjunto de datos pasado como primer argumento los elementos que también están contenidos en el segundo argumento. Como argumento opcional, esta función acepta una diferencia relativa máxima.

```
grouped = pyalps.groupSets(pyalps.flatten(data), ['J1', 'L'])
nd = []
for group in grouped:
    if group[0].props['Sz_total'] == 0:
        s0 = group[0]
        s1 = group[1]
    else:
        s0 = group[1]
        s1 = group[0]
    s0 = pyalps.subtract_spectrum(s0, s1)
    nd.append(s0)
    nd.append(s1)
data = nd
```

Ahora, creamos una nueva lista de conjuntos de datos (sector_E) que contendrá solo la energía del estado fundamental ('gs') o del primer estado excitado ('fe'). Almacenaremos esto en la propiedad which. Esto nos permitirá posteriormente usar la función `collectXY` para crear un gráfico de la energía del estado fundamental y del primer excitado frente al acoplamiento para cada L.

```
sector_E = []
grouped = pyalps.groupSets(pyalps.flatten(data), ['Sz_total', 'J1', 'L'])
for group in grouped:
    allE = []
    for q in group:
        allE += list(q.y)
    allE = np.sort(allE)
    
    d = pyalps.DataSet()
    d.props = dict_intersect([q.props for q in group])
    d.x = np.array([0])
    d.y = np.array([allE[0]])
    d.props['which'] = 'gs'
    sector_E.append(d)
    
    d2 = copy.deepcopy(d)
    d2.y = np.array([allE[1]])
    d2.props['which'] = 'fe'
    sector_E.append(d2)

sector_energies = pyalps.collectXY(sector_E, 'J1', 'Energy', ['Sz_total', 'which', 'L'])
plt.figure()
pyalps.plot.plot(sector_energies)
plt.xlabel('$J_1/J$')
plt.ylabel('$E_0$')
plt.legend(prop={'size':8})
```

En el último paso, calculamos el gap singlete y el gap triplete. Estos se definen como la diferencia de energía entre el estado más bajo del sistema y a) el primer estado excitado en el sector singlete ($S_z=0$), b) el estado más bajo en el sector triplete ($S_z=1$).

```
grouped = pyalps.groupSets( pyalps.groupSets(pyalps.flatten(data), ['J1', 'L']), ['Sz_total'])

gaps = []
for J1g in grouped:
    totalmin = 1000
    for q in flatten(J1g):
        totalmin = min(totalmin, np.min(q.y))
    
    for Szg in J1g:
        allE = []
        for q in Szg:
            allE += list(q.y)
        allE = np.sort(allE)
        d = pyalps.DataSet()
        d.props = pyalps.dict_intersect([q.props for q in Szg])
        d.props['observable'] = 'gap'
        print totalmin,d.props['Sz_total']
        if d.props['Sz_total'] == 0:
            d.y = np.array([allE[1]-totalmin])
        else:
            d.y = np.array([allE[0]-totalmin])
        d.x = np.array([0])
        d.props['line'] = '.-'
        gaps.append(d)

gaps = pyalps.collectXY(gaps, 'J1', 'gap', ['Sz_total', 'L'])

plt.figure()
pyalps.plot.plot(gaps)
plt.xlabel('$J_1/J$')
plt.ylabel('$\Delta$')
plt.legend(prop={'size':8})

plt.show()
```

## Cadena de Heisenberg con acoplamiento a segundos vecinos: asignaciones de TCC

Las correcciones de tamaño finito pueden reducirse significativamente ajustando a un punto crítico en una cadena frustrada J1-J2. A pesar de los distintos acoplamientos, se puede demostrar que este modelo tiene la misma teoría de campos crítica en el continuo, y por lo tanto podemos extraer las dimensiones de escala en este límite. Para una discusión detallada de este punto, consulte la referencia [I Affleck et al 1989 J. Phys. A: Math. Gen. 22 511](https://iopscience.iop.org/article/10.1088/0305-4470/22/5/015).

Compare con el espectro obtenido anteriormente: verá que la correspondencia con las dimensiones de escala esperadas es mucho más fácil de apreciar y que convergen mucho más rápido al aumentar el tamaño del sistema.
