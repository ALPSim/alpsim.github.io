En este tutorial examinaremos cadenas de espín críticas y estableceremos una conexión con su descripción en términos de teoría de campos conforme. En una transición de fase cuántica continua, una cadena de espín se vuelve invariante de escala y su física a larga distancia se describe mediante una teoría de campos conforme (CFT) $(1+1)$-dimensional. Una cadena finita nunca puede estar realmente en la transición, pero hereda una huella universal de ella: el espectro de baja energía de tamaño finito se organiza en torres etiquetadas por las dimensiones de escala $\Delta$ de los operadores primarios de la CFT y sus descendientes, con espaciados de nivel que se anulan como $1/L$. La diagonalización exacta es la herramienta ideal para observar esto directamente, ya que nos da el espectro completo de baja energía, incluidos sus momentos, para cadenas de unas pocas decenas de sitios —suficiente para resolver ya el contenido universal de operadores.

## Cadena de Ising

El primer modelo que consideraremos es la cadena de Ising crítica, dada por el hamiltoniano

$$
H=J_{z} \sum_{\langle i,j \rangle} S^i_z S^j_z + \Gamma \sum_i S^i_x
$$

Aquí, la primera suma recorre pares de primeros vecinos. $\Gamma$ se conoce como campo transverso; el sistema se vuelve crítico para $\Gamma/J=\frac{1}{2}$. Para $\Gamma=0$, el estado fundamental es antiferromagnético si $J\gt 0$ y ferromagnético si $J \lt 0$. El sistema es exactamente resoluble ([P. Pfeuty, Annals of Physics 57, 79 (1970)](https://doi.org/10.1016/0003-4916(70)90270-8)).

En la ecuación anterior, $\Delta$ se refiere a la dimensión de escala de ese campo. Los campos de escala aparecen en grupos: el más bajo, llamado campo primario, viene acompañado de un número infinito de descendientes con dimensión de escala $\Delta + m$, $m \in \lbrace 1, 2, 3, ... \rbrace$.

En la solución exacta del modelo de Ising (Ec. (3.7) en [el artículo de Pfeuty](https://doi.org/10.1016/0003-4916(70)90270-8)), se encuentra que las correlaciones de largo alcance decaen como:
$$
\langle S^i_z S^{i+n}_z \rangle \sim n^{-2\times 1/8}
$$
$$
\langle S^i_y S^{i+n}_y \rangle \sim n^{-2\times(1+1/8)}
$$
$$
\langle S^i_x S^{i+n}_x \rangle \sim n^{-2\times 1}
$$
Además, esperamos que la dimensión de escala del operador identidad sea 0.

Por lo tanto esperamos que aparezcan dimensiones de escala 0, 1/8, 1, 1+1/8 en la CFT del modelo de Ising. Para verlo, reescalaremos todas las energías del espectro según $E \rightarrow \frac{E-E_0}{(E_1-E_0)8}$. Esto forzará a que los dos estados más bajos caigan justo donde esperamos las dimensiones de escala; después podremos comprobar si el resto del espectro es consistente con esto.

### Parámetros

| Parámetro | Significado | Valor |
|---|---|---|
| `LATTICE` | cadena periódica | `chain lattice` |
| `MODEL` | modelo de espín cuántico | `spin` |
| `local_S` | número cuántico de espín por sitio | 1/2 |
| `Jxy` | acoplamiento en el plano ($S^xS^x+S^yS^y$) | 0 |
| `Jz` | acoplamiento de Ising ($S^zS^z$) | $-1$ |
| `Gamma` | campo transverso | 0.5 (el punto crítico, $\Gamma/|J_z|=1/2$) |
| `NUMBER_EIGENVALUES` | número de autoestados de baja energía solicitados a `sparsediag` | 5 |
| `L` | longitud de la cadena | 10, 12 |

Nótese que $S_z$ *no* se conserva una vez que $\Gamma\neq 0$ (el campo transverso invierte espines), por lo que, a diferencia de los tutoriales anteriores, no hay línea `CONSERVED_QUANTUMNUMBERS` y `sparsediag` diagonaliza la cadena completa, resuelta solo por el momento total.

### Red

La cadena de Ising crítica con campo transverso usa la misma `chain lattice` periódica de la [biblioteca de redes de ALPS](../../../documentation/intro/latticehowtos) que los tutoriales anteriores, ahora con un enlace de Ising $J_z$ y un campo transverso $\Gamma$ en cada sitio:

```
    Jz    Jz    Jz          Jz
o------o------o------ ... ------o     (each o also feels an on-site field Gamma S^x)
0      1      2                L-1
|______________________________________|
                Jz   (bond L-1 -- 0, periodic)
```

### Método

En $\Gamma=0.5$ la cadena no tiene $S_z$ conservado, por lo que la iteración de Lanczos de `sparsediag` se ejecuta sobre el espacio de Hilbert completo de dimensión $2^L$ (solo $1024$ y $4096$ estados para $L=10,12$); solicitar `NUMBER_EIGENVALUES=5` devuelve los cinco estados más bajos necesarios para ver el campo primario y sus primeros descendientes. Dado que solo necesitamos un puñado de estados de baja energía en lugar del espectro completo, la diagonalización dispersa de Lanczos sigue siendo mucho más económica que la diagonalización completa, aunque no se use ningún sector de simetría para reducir el tamaño del problema.

### Versión en Python

El archivo de parámetros para la simulación de Ising se puede encontrar <a href="../codes/ed-04-criticality/parm_ising" download>aquí</a>. La simulación se ejecuta con el script de Python.

Primero importaremos algunos módulos:

```
import pyalps
import pyalps.plot
import numpy as np
import matplotlib.pyplot as plt
import copy
import math
```
    
A continuación, configuremos los parámetros para dos tamaños de sistema. Tenga cuidado de usar el campo transverso $\Gamma$, no el campo longitudinal $h$.

```
parms = []
for L in [10,12]:
    parms.append({
        'LATTICE'    : "chain lattice",
        'MODEL'      : "spin",
        'local_S'    : 0.5,
        'Jxy'        : 0,
        'Jz'         : -1,
        'Gamma'      : 0.5,
        'NUMBER_EIGENVALUES' : 5,
        'L'          : L
    })
```

Como puede ver, simularemos dos tamaños de sistema. Ahora configuremos los archivos de entrada y ejecutemos la simulación:

```
prefix = 'ising'
input_file = pyalps.writeInputFiles(prefix,parms)
res = pyalps.runApplication('sparsediag', input_file)
# res = pyalps.runApplication('sparsediag', input_file, MPI=2, mpirun='mpirun')
data = pyalps.loadEigenstateMeasurements(pyalps.getResultFiles(prefix=prefix))
```

Descomentando la penúltima línea y ajustando el número de trabajos y el nombre de su ejecutable mpirun (que por defecto es mpirun), podría hacer que ALPS use varias CPU simultáneamente.

Primero, extraeremos el estado más bajo y el primer excitado para cada valor de L y los recopilaremos en un diccionario:

```
E0 = {}
E1 = {}
for Lsets in data:
    L = pyalps.flatten(Lsets)[0].props['L']
    allE = []
    for q in pyalps.flatten(Lsets):
        allE += list(q.y)
    allE = np.sort(allE)
    E0[L] = allE[0]
    E1[L] = allE[1]
```
        
El código anterior funciona porque sabemos que ALPS cargará las energías en listas agrupadas por simulación —data es por tanto una lista de listas donde tenemos distintas simulaciones en el nivel superior y distintos momentos debajo. Ahora reescalamos las energías según la ecuación dada arriba y recopilamos los datos en función de los momentos:

```
for q in pyalps.flatten(data):
    L = q.props['L']
    q.y = (q.y-E0[L])/(E1[L]-E0[L]) * (1./8.)
spectrum = pyalps.collectXY(data, 'TOTAL_MOMENTUM', 'Energy', foreach=['L'])
```
    
Para comparar, mostremos también los campos primarios y sus primeros descendientes:

```
for SD in [0.125, 1, 1+0.125, 2]:
    d = pyalps.DataSet()
    d.x = np.array([0,4])
    d.y = SD+0*d.x
    spectrum += [d]
```
        
Finalmente creamos el gráfico:

```
pyalps.plot.plot(spectrum)
plt.legend(prop={'size':8})
plt.xlabel("$k$")
plt.ylabel("E_0")
plt.xlim(-0.02, math.pi+0.02)
plt.show()
```

### Datos de salida

Diagonalizando independientemente el mismo hamiltoniano se obtienen los siguientes niveles de baja energía reescalados (sumados sobre momentos, ordenados) para las dos longitudes de cadena:

| $L$ | Nivel 1 | Nivel 2 | Nivel 3 | Nivel 4 |
|---|---|---|---|---|
| 10 | 0 (def.) | 0.125 (def.) | 0.994 | 1.107, 1.107 |
| 12 | 0 (def.) | 0.125 (def.) | 0.996 | 1.112, 1.112 |

Por construcción, los dos primeros niveles reescalados se sitúan exactamente en 0 y 0.125; el tercer y cuarto niveles son predicciones genuinas de la diagonalización y ya están, con estos tamaños tan pequeños, a pocos puntos porcentuales de los valores esperados de la CFT, 1 y 1.125, convergiendo aún más a medida que $L$ aumenta —evidencia numérica directa del contenido de operadores $\{0, \tfrac18, 1, 1+\tfrac18, \ldots\}$ de la CFT de Ising.

## Cadena de Heisenberg

Consideremos ahora un ejemplo más complicado: la cadena de Heisenberg antiferromagnética para grados de libertad de espín 1/2, descrita por
$$
H = J\sum_{\langle i,j \rangle} \mathbf{S}^i \cdot \mathbf{S}^j , \qquad J=1 .
$$

La teoría crítica de este modelo tiene una carga central $c=1$ con campos primarios 0, 0.5 y 1. A diferencia del modelo de Ising, las correcciones de tamaño finito se anulan solo logarítmicamente y por tanto son bastante pronunciadas en los tamaños de sistema que alcanzamos.

### Parámetros

| Parámetro | Significado | Valor |
|---|---|---|
| `LATTICE` | cadena periódica | `chain lattice` |
| `MODEL` | modelo de espín cuántico | `spin` |
| `local_S` | número cuántico de espín por sitio | 1/2 |
| `Jxy`, `Jz` | acoplamientos en el plano y de Ising (isótropos aquí) | 1, 1 |
| `NUMBER_EIGENVALUES` | autoestados solicitados por sector de momento | 5 |
| `CONSERVED_QUANTUMNUMBERS`, `Sz_total` | restringe al sector $S_z=0$, que contiene toda la torre de la CFT que necesitamos | `Sz`, 0 |
| `L` | longitud de la cadena | 10, 12 |

### Red

La misma `chain lattice` periódica de arriba (véase la [biblioteca de redes de ALPS](../../../documentation/intro/latticehowtos)), ahora con acoplamiento de Heisenberg isótropo y sin campo transverso:

```
    J     J     J           J
o-------o-------o--- ... ---o
0       1       2          L-1
|_________________________________|
                J   (bond L-1 -- 0, periodic)
```

### Método

A diferencia del punto de Ising, la cadena de Heisenberg conserva $S_z$, por lo que podemos restringir `sparsediag` al sector $S_z=0$ —este ya contiene el estado fundamental y suficiente parte de la torre de baja energía para leer el contenido de la CFT, a la vez que reduce la dimensión del sector de $2^{12}=4096$ a $\binom{12}{6}=924$ para $L=12$.

### Datos de salida

Diagonalizando de forma independiente, los niveles del sector $S_z=0$ reescalados por $(E-E_0)/\Delta_{S_z=1}\times\tfrac12$ (donde $\Delta_{S_z=1}=E_0(S_z{=}1)-E_0(S_z{=}0)$ es el gap triplete) resultan ser:

| $L$ | gap triplete $\Delta/J$ | Nivel 1 | Nivel 2 | Nivel 3 | Niveles 4, 5 |
|---|---|---|---|---|---|
| 10 | 0.4232 | 0 | 0.500 (def.) | 0.880 | 1.149, 1.149 |
| 12 | 0.3558 | 0 | 0.500 (def.) | 0.857 | 1.149, 1.149 |
| 14 | 0.3071 | 0 | 0.500 (def.) | 0.839 | 1.148, 1.148 |
| 16 | 0.2702 | 0 | 0.500 (def.) | 0.825 | 1.145, 1.145 |

El tercer nivel, que se espera que converja al descendiente de dimensión de escala 1, en cambio se aleja lentamente de él, pasando de 0.880 ($L=10$) a 0.825 ($L=16$), en lugar de aproximarse monótonamente como hicieron los niveles de Ising. Este comportamiento lento y no monótono es el síntoma práctico del operador marginalmente irrelevante mencionado arriba, cuyas correcciones suprimidas logarítmicamente dominan la dependencia con el tamaño en los valores de $L$ accesibles y solo se desvanecen en tamaños astronómicamente mayores. [ED-05](../ed05) muestra cómo, ajustando a un punto frustrado cercano, se elimina este operador marginal y se recupera una convergencia rápida y de libro de texto en $1/L$.

## Resumen

Tanto la cadena de Ising con campo transverso, exactamente resoluble, como la cadena de Heisenberg, de convergencia más difícil, confirman, únicamente a partir de espectros de diagonalización exacta de tamaño finito, el contenido universal de operadores de la CFT ($c=1/2$ con dimensiones $\{0,\tfrac18,1,\ldots\}$ para Ising; $c=1$ con dimensiones $\{0,\tfrac12,1,\ldots\}$ para Heisenberg) predicho por sus respectivas teorías de campos —el caso de Heisenberg ilustra además cómo un operador marginalmente irrelevante puede hacer que esa convergencia sea engañosa numéricamente.
