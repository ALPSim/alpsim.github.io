El objetivo de este tutorial es calcular propiedades termodinámicas de sistemas de espín cuánticos usando el algoritmo de Wang-Landau cuántico (QWL).
A diferencia de los métodos QMC de integral de camino, que simulan a una temperatura fija, QWL realiza un paseo aleatorio en el espacio de órdenes de expansión de la función de partición para estimar directamente la densidad de estados.
A partir de una sola simulación se pueden entonces evaluar observables termodinámicos — energía, calor específico, entropía, susceptibilidad — en todo el rango de temperaturas sin volver a ejecutar la simulación.

## Termodinámica de cadenas de espín de Heisenberg cuánticas

### La cadena de Heisenberg ferromagnética

El archivo de parámetros <a href="https://github.com/ALPSim/ALPS/blob/master/tutorials/mc-06-qwl/parm6a" download>`parm6a`</a> configura una simulación QWL del ferromagneto de Heisenberg $S=\frac{1}{2}$ en una cadena de 40 sitios:

```
LATTICE="chain lattice"
MODEL="spin"
local_S = 1/2
L       = 40
CUTOFF  = 500
{J = -1}
```

`CUTOFF` establece el orden máximo de expansión de la función de partición muestreado por el algoritmo; 500 es suficiente para una cadena de 40 sitios en el rango de temperaturas estudiado aquí.

### Línea de comandos

Después de preparar los archivos de entrada y ejecutar el código `qwl`:

```
parameter2xml parm6a
qwl parm6a.in.xml
```

genere los archivos XML de gráficos para todos los observables termodinámicos usando `qwl_evaluate`:

```
qwl_evaluate --T_MIN 0.1 --T_MAX 10 --DELTA_T 0.1 parm6a.task1.out.xml
```

Esto genera un archivo XML por observable:

```
parm6a.task1.plot.energy.xml
parm6a.task1.plot.free_energy.xml
parm6a.task1.plot.entropy.xml
parm6a.task1.plot.specific_heat.xml
parm6a.task1.plot.uniform_structure_factor.xml
parm6a.task1.plot.staggered_structure_factor.xml
parm6a.task1.plot.uniform_susceptibility.xml
```

Extraiga los datos a texto plano con `plot2text`:

```
plot2text parm6a.task1.plot.energy.xml
```

Las herramientas `plot2xmgr` (Grace) y `plot2gp` (Gnuplot) producen salidas equivalentes en esos formatos. El método preferido de análisis es Python, descrito a continuación.

### Python

El script <a href="https://github.com/ALPSim/ALPS/blob/master/tutorials/mc-06-qwl/tutorial6a.py" download>`tutorial6a.py`</a> configura y ejecuta la simulación, y luego evalúa todos los observables en una sola llamada:

```python
import pyalps
import matplotlib.pyplot as plt

data = pyalps.evaluateQWL(pyalps.getResultFiles(prefix='parm6a'),
                          DELTA_T=0.1, T_MIN=0.1, T_MAX=10.0)

for s in pyalps.flatten(data):
    plt.figure()
    plt.title("Ferromagnetic Heisenberg chain")
    pyalps.plot.plot(s)
    plt.show()
```

Para el ferromagneto ($J=-1$) debería verse un amplio pico de calor específico a baja temperatura (una anomalía tipo Schottky) y una susceptibilidad uniforme que diverge cuando $T\to 0$, en concordancia con el desarrollo de orden ferromagnético a temperatura cero en 1D.

### La cadena de Heisenberg antiferromagnética

Para simular la cadena antiferromagnética, fije $J=1$ en lugar de $J=-1$.
Los parámetros están en <a href="https://github.com/ALPSim/ALPS/blob/master/tutorials/mc-06-qwl/parm6b" download>`parm6b`</a> y el script de Python en <a href="https://github.com/ALPSim/ALPS/blob/master/tutorials/mc-06-qwl/tutorial6b.py" download>`tutorial6b.py`</a>; todo lo demás es idéntico al caso ferromagnético.

Para el antiferromagneto, la susceptibilidad uniforme permanece finita cuando $T\to 0$ (una firma del estado fundamental de líquido de espín del antiferromagneto 1D), mientras que el pico de calor específico se desplaza y ensancha de forma distinta.

## El antiferromagneto de Heisenberg tridimensional

### Simulación del antiferromagneto de Heisenberg cuántico 3D

El archivo de parámetros <a href="https://github.com/ALPSim/ALPS/blob/master/tutorials/mc-06-qwl/parm6c" download>`parm6c`</a> configura una simulación QWL del antiferromagneto de Heisenberg $S=\frac{1}{2}$ en una red cúbica simple con $4^3=64$ sitios.
El script de Python es <a href="https://github.com/ALPSim/ALPS/blob/master/tutorials/mc-06-qwl/tutorial6c.py" download>`tutorial6c.py`</a>.
Ejecute y evalúe exactamente como para la cadena anterior.

El factor de estructura alterno $S(\pi,\pi,\pi)$ debería comenzar a aumentar abruptamente cerca de $T\approx 1$, señalando el inicio de las correlaciones antiferromagnéticas.
El calor específico muestra un pico correspondiente y la entropía disminuye rápidamente en el mismo rango de temperatura.

### Análisis de escalamiento de tamaño finito para determinar el punto crítico

El escalamiento de tamaño finito predice que el factor de estructura alterno en el punto crítico escala como $S(L) \propto L^{2-\eta}$ con $\eta \approx 0.034$ (clase de universalidad de Heisenberg clásico 3D).
Un gráfico de $S(L)/L^{2-\eta}$ frente a la temperatura debería mostrar curvas para distintos $L$ que se cruzan en la temperatura crítica $T_c$.

El archivo de parámetros <a href="https://github.com/ALPSim/ALPS/blob/master/tutorials/mc-06-qwl/parm6d" download>`parm6d`</a> (o <a href="https://github.com/ALPSim/ALPS/blob/master/tutorials/mc-06-qwl/tutorial6d.py" download>`tutorial6d.py`</a>) ejecuta el antiferromagneto cúbico para dos tamaños de sistema ($L=4$ y $L=6$) con un `CUTOFF=1000` mayor para mantener la precisión a temperaturas más bajas.
Después de ejecutar, cargue los resultados:

```python
import pyalps
import matplotlib.pyplot as plt
import copy

results = pyalps.evaluateQWL(pyalps.getResultFiles(prefix='parm6d'),
                             DELTA_T=0.05, T_MIN=0.5, T_MAX=1.5)
```

Extraiga el factor de estructura alterno para cada tamaño de sistema, reescálelo por $L^{-(2-\eta)}$, y etiquételo por $L$:

```python
eta = 0.034
data = []
for s in pyalps.flatten(results):
    if s.props['ylabel'] == 'Staggered Structure Factor per Site':
        d = copy.deepcopy(s)
        l = s.props['L']
        d.props['label'] = 'L=' + str(l)
        d.y = d.y * pow(float(l), -(2 - eta))  # rescale by L^{-(2-eta)}
        data.append(d)
```

Luego grafique las curvas de escalamiento:

```python
plt.figure()
plt.title("Scaling plot for cubic lattice Heisenberg antiferromagnet")
pyalps.plot.plot(data)
plt.legend()
plt.xlabel('Temperature $T/J$')
plt.ylabel('$S(\pi,\pi,\pi)\, L^{-(2-\eta)}$')
plt.show()
```

Las curvas para distintos $L$ deberían cruzarse cerca de $T_c \approx 0.946$.

## Preguntas

- Compare el calor específico y la susceptibilidad uniforme de las cadenas ferromagnética ($J=-1$) y antiferromagnética ($J=1$). ¿Dónde son más pronunciadas las diferencias, y por qué los dos casos son casi idénticos a alta temperatura?
- ¿Cuál es la entropía de la cadena en $T=0$ y cuando $T\to\infty$? (Si los datos de $L=40$ son ruidosos a baja $T$, repita con $L=8$.) ¿Es esto consistente con la tercera ley de la termodinámica?
- ¿Por qué diverge la susceptibilidad uniforme cuando $T\to 0$ para el ferromagneto pero permanece finita para el antiferromagneto?
- ¿Por qué comienza a aumentar el factor de estructura alterno del antiferromagneto 3D cerca de $T\approx 1$? ¿Qué otras firmas termodinámicas acompañan esto?
- ¿Se cruzan las curvas reescaladas del factor de estructura $S(L)\,L^{-(2-\eta)}$ en una sola temperatura? ¿Cuál es su estimación de $T_c$? Compárela con el valor de la literatura $T_c = 0.946$.
- ¿Cómo podría obtener una estimación más precisa de $T_c$? ¿Qué necesitaría cambiar en la simulación?
- ¿Esperaría que la temperatura crítica del *ferromagneto* de Heisenberg cuántico en una red cúbica fuera la misma que para el antiferromagneto? ¿Cómo lo comprobaría?
