Más allá de los promedios termodinámicos como la energía y la magnetización, las simulaciones QMC pueden medir funciones de correlación espacial y sus transformadas de Fourier.
Este tutorial demuestra estas mediciones usando el código SSE de bucle dirigido en el modelo de Heisenberg $S = 1/2$ sobre una red cuadrada pequeña.

La función de correlación espín-espín $C(\mathbf{r}) = \langle S^z_0 S^z_\mathbf{r} \rangle$ describe cómo se correlacionan las orientaciones de espín en dos sitios separados por $\mathbf{r}$.
Para el modelo de Heisenberg antiferromagnético a baja temperatura, las correlaciones alternan de signo con la distancia, reflejando la tendencia hacia el orden de Néel.
El factor de estructura $S(\mathbf{q}) = \sum_\mathbf{r} e^{i\mathbf{q}\cdot\mathbf{r}} C(\mathbf{r})$ es la transformada de Fourier de las correlaciones; alcanza su máximo en el vector de onda antiferromagnético $\mathbf{q} = (\pi, \pi)$.

## Red cuadrada de Heisenberg bidimensional

Simulamos el modelo de Heisenberg $S = 1/2$ en una red cuadrada de $4 \times 4$ a temperatura $T = 0.3$ con un pequeño campo $h = 0.1$ para romper débilmente la simetría de rotación de espín.
El tamaño pequeño del sistema permite una simulación rápida; los efectos de tamaño finito son significativos a esta escala.

### Línea de comandos

El archivo de parámetros <a href="https://github.com/ALPSim/ALPS/blob/master/tutorials/mc-04-measurements/parm4" download>`parm4`</a> habilita tres indicadores de medición adicionales junto con los parámetros estándar:

```
MODEL="spin"
LATTICE="square lattice"
local_S=1/2
MEASURE[Correlations]=true
MEASURE[Structure Factor]=true
MEASURE[Green Function]=true
THERMALIZATION=10000
SWEEPS=500000
J=1
L=4
W=4
T=0.3
{h=0.1;}
```

`MEASURE[Correlations]` registra $\langle S^z_i S^z_j \rangle$ para todos los pares de sitios.
`MEASURE[Structure Factor]` registra $S(\mathbf{q})$ en todos los vectores de onda compatibles con la red.
`MEASURE[Green Function]` registra la función de Green en tiempo imaginario, que puede continuarse analíticamente para obtener información espectral.

Convierta y ejecute:

```
parameter2xml parm4
dirloop_sse --Tmin 10 --write-xml parm4.in.xml
```

`--Tmin 10` establece el intervalo de checkpoint en 10 segundos; `--write-xml` escribe archivos de salida XML que pyalps puede leer junto con los resultados HDF5.

### Python

El script <a href="https://github.com/ALPSim/ALPS/blob/master/tutorials/mc-04-measurements/tutorial4.py" download>`tutorial4.py`</a>:

```Python
import pyalps

parms = [{
    'LATTICE'                   : "square lattice",
    'MODEL'                     : "spin",
    'MEASURE[Correlations]'     : True,
    'MEASURE[Structure Factor]' : True,
    'MEASURE[Green Function]'   : True,
    'local_S'                   : 0.5,
    'T'                         : 0.3,
    'J'                         : 1,
    'THERMALIZATION'            : 10000,
    'SWEEPS'                    : 500000,
    'L'                         : 4,
    'W'                         : 4,
    'h'                         : 0.1
}]

input_file = pyalps.writeInputFiles('parm4', parms)
pyalps.runApplication('dirloop_sse', input_file, Tmin=10)
```

### Evaluación de los resultados

Cargue todas las mediciones desde los archivos de salida e imprímalas:

```Python
import pyalps

data = pyalps.loadMeasurements(pyalps.getResultFiles(prefix='parm4'))

for s in pyalps.flatten(data):
    if len(s.x) == 1:
        print(s.props['observable'], ':', s.y[0])
    else:
        for (x, y) in zip(s.x, s.y):
            print(s.props['observable'], x, ':', y)
```

Los observables escalares (energía, magnetización, susceptibilidad) se imprimen en una sola línea.
Los observables vectoriales (correlaciones, factor de estructura, función de Green) se imprimen una entrada por línea, con su índice — par de sitios, vector de onda o punto de tiempo imaginario — junto al valor.

### Graficado de las correlaciones espín-espín

Para visualizar la función de correlación, cargue el observable `Sz Correlations` y grafíquelo:

```Python
import pyalps
import matplotlib.pyplot as plt
import pyalps.plot

data = pyalps.loadMeasurements(pyalps.getResultFiles(prefix='parm4'), 'Sz Correlations')
correlations = pyalps.collectXY(data, x='d', y='Sz Correlations')

plt.figure()
pyalps.plot.plot(correlations)
plt.xlabel('Distance $r$')
plt.ylabel('$\\langle S^z_0 S^z_r \\rangle$')
plt.title('Spin correlations, $4\\times 4$ Heisenberg model')
plt.show()
```

Las correlaciones deberían alternar en signo con la distancia, con el valor de primeros vecinos negativo (antiferromagnético) y su magnitud disminuyendo con $r$.

### Graficado del factor de estructura

```Python
import pyalps
import matplotlib.pyplot as plt
import pyalps.plot

data = pyalps.loadMeasurements(pyalps.getResultFiles(prefix='parm4'), 'Sz Structure Factor')
structure_factor = pyalps.collectXY(data, x='q', y='Sz Structure Factor')

plt.figure()
pyalps.plot.plot(structure_factor)
plt.xlabel('Wavevector $q$')
plt.ylabel('$S^z(q)$')
plt.title('Structure factor, $4\\times 4$ Heisenberg model')
plt.show()
```

El valor más grande debería aparecer en el vector de onda antiferromagnético $\mathbf{q} = (\pi, \pi)$, correspondiente al patrón alterno de espines del orden de Néel.
