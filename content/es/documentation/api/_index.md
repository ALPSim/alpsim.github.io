Este manual contiene un resumen de las funciones de `pyalps` utilizadas en los tutoriales de ALPS. La sección de la API de Python está estructurada de la siguiente manera:

- [Ejecutar aplicaciones](runapp): Funciones para ejecutar aplicaciones de ALPS
- [Carga de datos](loadat): Funciones para cargar mediciones generadas por aplicaciones específicas
- [Evaluación](eval): Funciones genéricas para realizar análisis de datos, p. ej. recopilar resultados y graficarlos.

Consulte los [tutoriales de aplicaciones de ALPS](../../../../tutorials) para ver cómo se emplean estas funciones en simulaciones reales. Por ejemplo: cómo recorrer varias temperaturas y graficar la magnetización promedio en función de la temperatura, o cómo extrapolar efectos de tamaño finito.
