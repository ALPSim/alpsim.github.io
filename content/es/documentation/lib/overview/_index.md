Las bibliotecas ALPS forman parte del proyecto ALPS (Parallel Algorithms for Lattice Models), que tiene como objetivo proporcionar algoritmos paralelos genéricos para modelos de red clásicos y cuánticos, además de ofrecer clases de utilidad y algoritmos para muchos otros problemas. Su propósito es aumentar la reutilización de software en la comunidad de la física.

## Bibliotecas

Las siguientes bibliotecas están disponibles públicamente como parte del proyecto de bibliotecas ALPS:

- [alps/general](../general): clases y traits de uso general
- [alps/random](../random): extensiones a las bibliotecas de números aleatorios de Boost y C++.
- [alps/parser](../parser): funciones auxiliares para analizar y escribir archivos XML
- [alps/alea](../alea): mediciones y evaluación de Monte Carlo
- [alps/graph](../bgl): extensiones a la biblioteca Boost Graph
- [alps/hdf5](../hdf5): una biblioteca para leer y escribir archivos HDF5 desde C++

Además de las herramientas auxiliares para la biblioteca del planificador (scheduler), hemos proporcionado algunas herramientas de ejemplo para la manipulación de los archivos XML creados por los programas de ALPS.
